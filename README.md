## ✨ Features

- **Multi-Pile Booking Architecture**: Allows users to segregate items into distinct services (e.g., Wash & Fold by kg vs. Dry Clean per piece) within a single order to prevent incompatible processing.
- **Wash Preferences & Add-ons**: Built-in support for selecting premium detergents and fabric softeners seamlessly at checkout.
- **Robust Payment Logic (Buffer Hold)**: Employs a modern ~15% authorization buffer to handle final weight variances without friction at delivery.
- **Unified Order Tracking & History**: Features an interactive dashboard tracking active services step-by-step and rendering itemized receipts for historical past orders.
- **Consistent Design System**: The UI follows a strict 8px spatial grid for perfect visual rhythm and layout consistency.
- **Dynamic Experience**: Fluid data flow handling user details, addresses, and active orders.
- **Accessible Design**: Built with clear contrasts and visible focus states for ease of use.
